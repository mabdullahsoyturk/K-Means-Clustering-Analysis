# Analysis of K Means Clustering written in Python and C

### Build

```sh
$ git clone https://github.com/mabdullahsoyturk/K-Means-Clustering-Analysis.git
$ cd K-Means-Clustering-Analysis
$ make
```

### Run
```sh
$ ./output
```

# Running times with Iris Dataset

| Python | C |
| ------ | ------ |
| 29.3 ms | 0.245 ms |
